# World Of Computing

Autonomous Codex workspace for writing **Next Token: The Race to Build the Machines That Learned Language, Code, and Computing**.

Start the loop with:

```text
/goal Read GOAL.md fully, then run the loop forever per its protocol.
```

The repository tracks source files, manuscript drafts, ledgers, lightweight chart/data assets, and loop memory. Heavy rendered PDFs and raster/media assets are intentionally excluded from Git; their provenance belongs in `assets_manifest.tsv`.
