# Chapter 18 Tools/Agents Visual Package

Status: promoted visual package, pass I-0120, 2026-05-26.

This pass adds four lightweight, rights-safe SVGs for Chapter 18:

- A-0060, `chapter18-rag-evidence-conveyor.svg`: RAG as an evidence conveyor with visible failure gates.
- A-0061, `chapter18-function-call-boundary.svg`: function calling as an untrusted model proposal crossing into application validation and permission checks.
- A-0062, `chapter18-agent-loop-harness.svg`: agent behavior as a harness-mediated loop rather than model-only autonomy.
- A-0063, `chapter18-prompt-injection-boundary.svg`: prompt injection as an instruction/data boundary risk.

The package follows the Chapter 18 claim audit: visuals can explain mechanisms and control surfaces, but they cannot claim that retrieval guarantees truth, function calls are safe commands, agent loops are reliable autonomous workers, MCP/computer-use proves adoption or productivity, or prompt injection has a quantified prevalence. The figures are designed as claim-control diagrams, not decorative agent art.
