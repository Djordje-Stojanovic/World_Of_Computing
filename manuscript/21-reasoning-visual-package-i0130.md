# Chapter 21 Reasoning Visual Package

Status: promoted visual package, pass I-0130, 2026-05-26.

This package adds four rights-safe SVGs for Chapter 21:

- `A-0078` - Train-time versus test-time compute axes.
- `A-0079` - Verifier/search loop with candidate paths and observations.
- `A-0080` - Inference-contract benchmark matrix.
