# Chapter 23 Trust Visual Package

Status: promoted visual package, pass I-0129, 2026-05-26.

This package adds four rights-safe SVGs for Chapter 23:

- `A-0074` - Failure modes are different claims: hallucination/truth, sycophancy, adversarial, proxy, evidence, and measurement failures.
- `A-0075` - Trust is a stack: evidence selection, evidence use, authority, evaluation, permission/action, and audit memory.
- `A-0076` - Every evaluation casts a shadow: TruthfulQA, Arena preference, SWE-bench, and LiveCodeBench as useful but scoped instruments.
- `A-0077` - From answer to auditable claim: a provenance workflow that separates generation from supported publication.

The figures use only original lightweight vector geometry and source IDs from `data/chapter23_trust_claim_audit_i0128.tsv` and `data/chapter23_trust_source_pack_i0128.tsv`. They intentionally block exact hallucination rates, incident prevalence, deployed-product sycophancy, current jailbreak exploitability, prompt-injection prevalence, production safety, hidden behavior, mitigation effectiveness, live ranks, productivity, and global trust-score claims.
