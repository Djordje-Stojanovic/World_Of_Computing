# Chapter 20 Coding-Agent Visual Package

Status: promoted visual package pass I-0108, 2026-05-25.

This package adds four lightweight SVG visuals for Chapter 20. The visuals are designed as claim-control surfaces: they make the coding-agent work loop legible while blocking benchmark triumphalism, product-superiority claims, and deployed-productivity inferences.

## Visual Sequence

1. `A-0032` - Coding Agent Harness Loop: shows issue, context pack, model, tools, checks, retries, and review as one measured system.
2. `A-0033` - Coding Benchmark Caveat Matrix: separates SWE-bench, LiveCodeBench, terminal tasks, and product metrics by allowed use and blocked leaps.
3. `A-0034` - Tool-Call Lifecycle: shows proposal, permission, execution, observation, retry, and human review.
4. `A-0035` - Productivity Claim Blocker Map: shows which Chapter 20 claims are allowed now and which require customer-side or independent outcome evidence.
