# Physical Photo Rights Hunt - I-0181

Pass I-0181 creates a rights-first hunt list for physical images: lithography/ASML proxy, fab cleanroom, wafer, server rack, data-center hall and exterior, substation, transmission lines, gas turbine, battery storage, cooling tower, and nuclear plant.

The package does not commit image files. It records candidate source pages, private target paths, license-proof fields, alt captions, and blocked claims so a later capture pass can download only images whose reuse rights and attribution strings are durable.

## Strongest Candidates

- Physical manufacturing texture: cleanroom, wafer, and EUV/lithography proxy.
- Compute texture: NERSC/server racks and a generic data-center hall.
- Power texture: substation, transmission, gas turbine, battery storage, and nuclear/cooling tower imagery.

## Weakest Candidate

The ASML-machine need remains only partially satisfied. A licensed ASML-specific machine photograph is still the ideal target; until then, the safe path is to use a licensed lithography-tool or ASML-building proxy and caption it as proxy evidence, not machine evidence.

## Caption Contract

These photos can show physical scale and material constraints. They cannot prove production capacity, supply allocation, GPU counts, data-center workloads, power delivery, emissions, grid capacity, operational status, or any direct link between the pictured object and a named LLM system unless the same claim has separate same-scope evidence.
