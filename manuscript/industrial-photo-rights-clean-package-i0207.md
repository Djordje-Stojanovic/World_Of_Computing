## Caption Firewall

The safe industrial-photo role is atmosphere plus physical specificity: machines have scale, cleanrooms have procedures, racks and cooling equipment occupy real rooms, and the grid has visible hardware. These images must not silently carry claims about named customers, AI workloads, supplier share, foundry node, yield, power delivery, emissions, site capacity, or production allocation.

## Use Guidance

For final layout, the strongest low-risk package is a spread that combines one lithography/cleanroom/wafer manufacturing image, one rack/cooling image, and one grid/storage image. The page should label these as infrastructure texture rather than evidence for a named company's proprietary buildout.

The unresolved rows should be treated as art-direction instructions, not approved assets. If a specific transmission-line, gas-turbine, or cooling-tower replacement is selected later, add a fresh manifest row or a final-license row rather than mutating this audit result.
