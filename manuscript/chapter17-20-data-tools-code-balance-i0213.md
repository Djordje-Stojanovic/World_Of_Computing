# Chapter 17-20 Data/Tools/Code Exhibit Balance - I-0213

Pass I-0213 audits the Chapter 17-20 exhibit bench as one rising system rather than four isolated technical topics. The intended reader path is:

1. Chapter 17: data becomes material and governed. Tokens, corpus filters, mixture recipes, and contamination blockers show that "data" is a supply chain, not a neutral pile.
2. Chapter 18: data becomes selected context and bounded action. Retrieval, function calls, and agent harnesses show the trust membrane before agency is named.
