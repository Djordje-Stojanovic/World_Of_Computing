# Private-Use Screenshots

Screenshots saved here are private-use raster evidence and are ignored by Git through `assets/**/*.png` and related image patterns.

Commit only:

- queue rows
- capture log rows
- hashes and dimensions
- source URL/title/viewport/timestamp
- rights/fair-use notes
- manifest rows or manifest drafts

Do not force-add screenshot rasters unless the user explicitly changes the project policy.

I-0281 convention: product pages, docs, leaderboards, and model-card screenshots must record `source_url_or_local_path`, `accessed_at`, `capture_tool`, `viewport`, `local_path`, `sha256`, `rights_status`, `story_purpose`, and `blocked_claims` before they enter an exhibit manifest.
