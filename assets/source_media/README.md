# Ignored Source Media

Heavy source media such as PDFs, slide renders, large screenshots, archives, and downloaded originals can be stored locally here when needed for private analysis.

Git policy:

- media files stay ignored
- hashes, page refs, dimensions, and notes are committed in `data/`
- final art promotion requires a manifest row and rights/quote review

This directory is for local evidence handling, not publication clearance.

I-0281 convention: downloaded PDFs, HTML, raw captures, and page-source mirrors must record `source_url`, `source_id`, `access_date`, `file_size`, `sha256`, `content_type`, `quote_limit`, and `blocked_claims`.
