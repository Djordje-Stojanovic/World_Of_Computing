# Rights And Quote Reviews

Rights and quote review rows live here. Start from `data/capture_templates/rights_quote_review_template.tsv`.

Use this folder before promoting any screenshot, source card, photo, extracted slide, or direct quote into final-art consideration.

Every selected final exhibit needs:

- rights status or private-use/fair-use note
- creator or organization
- source URL/path and access date
- quote limit or no-quote note
- caption blocker text
- publication action: publish, redraw, replace, permission-needed, fair-use-review, or drop
